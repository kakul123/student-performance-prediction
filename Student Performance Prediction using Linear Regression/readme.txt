# 🎓 Student Performance Predictor

A Streamlit web app to predict student performance based on study hours, previous scores, sleep, extracurricular activities, and practice.

## 🔧 Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (Linear Regression)
- Streamlit
- Matplotlib, Seaborn

## 🚀 How to Run
1. Clone the repo  
2. Install dependencies:  
   `pip install -r requirements.txt`
3. Run the app:  
   `streamlit run main.py`

## 📂 Input Features
- Hours Studied  
- Previous Scores  
- Sleep Hours  
- Extracurricular Activities  
- Practice Papers Solved

## 📈 Output
Predicted **Performance Index** with feature importance and visual analysis.
